
# LLM Models Guide

This section explains the models that can be used with Ollama.

Different models have different purposes.

Some are optimized for:

• conversation  
• reasoning  
• coding  
• lightweight systems

---

# Model Categories

Models generally fall into these categories:

General Models  
Coding Models  
Lightweight Models  
Reasoning Models

---

# TinyLlama

TinyLlama is a **very small language model**.

Size: ~600MB

Advantages:

• extremely lightweight  
• runs on low RAM systems  
• fast

Disadvantages:

• limited reasoning ability

Recommended for:

• testing  
• low-power machines

---

# Mistral

Mistral is a **general purpose model**.

Size: ~4GB

Advantages:

• strong conversational ability  
• good reasoning  
• widely used

Recommended for:

• chat  
• learning LLM usage  
• general AI tasks

---

# Phi-3

Phi-3 is a **small but powerful reasoning model** developed by Microsoft.

Advantages:

• strong reasoning  
• smaller than many models  
• efficient

Recommended for:

• reasoning tasks  
• logic problems  
• structured outputs

---

# DeepSeek Coder

DeepSeek Coder is optimized for programming.

Advantages:

• excellent code generation  
• good debugging capability

Recommended for:

• developers  
• programming tasks

---

# Choosing the Right Model

If you are a beginner:

Start with:

tinyllama

Then upgrade to:

mistral

If you want coding help:

deepseek-coder

If you want strong reasoning:

phi3
