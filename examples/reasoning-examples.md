
# Reasoning Examples

Reasoning models are good at solving logical problems step-by-step.

Examples include **Phi-3 Mini** and **Mistral**.

---

# Example 1 — Math Problem

Prompt:

```
If a train travels 60 km in 1 hour, how far will it travel in 3 hours?
Explain step by step.
```

Expected reasoning:

```
Speed = 60 km/hour

Distance = Speed × Time

Distance = 60 × 3

Distance = 180 km
```

---

# Example 2 — Logical Puzzle

Prompt:

```
A farmer has 17 sheep. All but 9 run away. How many remain?
```

Correct reasoning:

```
All but 9 ran away.

So 9 sheep remain.
```

---

# Example 3 — Step-by-Step Explanation

Prompt:

```
Explain how binary search works step by step.
```

Expected output:

1. Find the middle element
2. Compare with target
3. Eliminate half of the array
4. Repeat until found
