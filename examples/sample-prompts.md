
# Prompt Examples

This section demonstrates how to interact with local AI models using effective prompts.

A **prompt** is the instruction you give to the AI model.

Better prompts lead to better responses.

---

# Basic Prompt Example

Prompt:

```
Explain what a Large Language Model is.
```

Example response:

```
A Large Language Model (LLM) is an artificial intelligence system trained on large amounts of text data to understand and generate human language.
```

---

# Instruction Prompt

Prompt:

```
Explain neural networks in simple terms for a beginner.
```

Purpose:

* beginner-friendly explanation
* simplified language

---

# Step-by-Step Prompt

Prompt:

```
Explain how machine learning works step by step.
```

Why this works:

Models perform better when asked to **break answers into steps**.

---

# List Generation Prompt

Prompt:

```
List the advantages of running AI models locally.
```

Expected result:

* privacy
* offline usage
* no API costs
* full control

---

# Structured Output Prompt

Prompt:

```
Explain supervised learning in 5 bullet points.
```

This forces the model to generate structured output.

---

# Teaching Prompt

Prompt:

```
Act as a teacher and explain Python loops to a beginner.
```

This technique is called **role prompting**.

---

# Best Practices for Prompting

Good prompts usually include:

• clear instructions
• context
• output format

Example:

```
Explain Docker in simple language using bullet points.
```
