
# Model Storage Options

LLM models are large.

Many models range between **2GB and 10GB**.

Therefore it is important to manage storage carefully.

---

# Default Storage Location

Ollama stores models in:


~/.ollama/models


This location is inside the user home directory.

---

# Problem With Default Storage

If you install many models:

• system disk becomes full  
• OS performance decreases

---

# External Storage Solution

You can store models on:

• USB drives  
• external SSD  
• external HDD

Example:


/media/user/USB/LLMS


---

# Benefits of External Storage

Advantages:

• saves internal disk space  
• portable models  
• easy to expand storage

---

# When To Use External Storage

Use external storage when:

• installing many models  
• system disk is small  
• working with large models
